How It Works.

From raw video to processed output in under 5 minutes. No setup, no configuration files, no command line.

01
Download & Run the EXE
Get the ArNova VideoProcessor.exe from the admin. Double-click to launch — no Python, no FFmpeg install, no dependencies needed. Everything is bundled inside.
ArNova VideoProcessor.exe ← just double-click
02
Browse & Select Your Video
Click the Browse button, select any .mp4 file on your machine. The full path fills in automatically.
03
Configure Timer
Toggle the timer on or off. Choose whether it appears in the top-left or top-right corner of the video using the pill selector.
04

Paste a GIF URL (or local path). Set the exact second it should appear (Start) and disappear (End). Set X/Y position on screen, physical size in cm, and animation FPS.

Enter any multiplier. 1.0 keeps original speed. 2.0 doubles speed. 0.5 slows it down. The cursor stays perfectly visible even at high speeds.

007 — Questions

Frequently Asked.

Do I need Python or FFmpeg installed? ▼
No. The EXE bundles everything — Python runtime, OpenCV, Pillow, FFmpeg — all inside one file. Just double-click and run on any Windows 10/11 machine.
What video formats are supported? ▼
The tool works with MP4 files. Output is always H.264 MP4 compressed with FFmpeg for maximum compatibility with all players and platforms.
How do I get the cursor GIF to show at the right time? ▼
Set the Start (s) and End (s) fields to the exact seconds in the video. For example: Start 0.36, End 43.0 shows the cursor from 0.36 seconds to 43 seconds. The timer overlay on the video helps you identify the exact timestamps first.
What happens after my 4 free credits run out? ▼
A popup appears with your admin's WhatsApp and email. Contact the admin to get more credits unlocked instantly. Credits are tracked locally on your machine using the Windows registry.
Can I use a local GIF instead of a URL? ▼
Yes. Paste the full local path to the GIF file in the GIF URL/Path field, e.g. C:\Users\you\Downloads\cursor.gif — it works exactly the same as a URL.
How big is the output file? ▼
FFmpeg compresses the output with CRF 23 (high quality, small size). A typical 1-minute screen recording compresses to 5–20 MB depending on content complexity.
Does it work on macOS or Linux? ▼
Currently the EXE is Windows-only. The Python source code runs on macOS/Linux if you install the dependencies manually. Contact the admin if you need a cross-platform build.
